Exercise 19 – Customization with Sass

Override Bootstrap's default variables using Sass to create a custom theme.

Exercise 19.1 – Setting Up Bootstrap 5 with Sass

Step 1 – Initialize your project
mkdir my-bootstrap-project
cd my-bootstrap-project
npm init -y
npm install bootstrap sass
Step 2 – Create folder structure
my-bootstrap-project/
├── scss/
│   └── custom.scss      ← your custom Sass file
├── css/
│   └── custom.css       ← compiled output (auto-generated)
├── js/
│   └── (copy bootstrap.bundle.min.js here)
└── index.html
Step 3 – Add a compile script to package.json
// package.json
{
  "scripts": {
    "sass": "sass --watch scss/custom.scss css/custom.css"
  }
}

// Run with:
npm run sass
Step 4 – Link the compiled CSS in index.html
<link rel="stylesheet" href="css/custom.css"/>
<script src="js/bootstrap.bundle.min.js"></script>

Exercise 19.2 – Customising Variables via _variables.scss

The key rule: override variables BEFORE importing Bootstrap.

scss/custom.scss
// ============================================================
// STEP 1 — Override Bootstrap variables (BEFORE the import)
// ============================================================

// Change primary colour
$primary: #e74c3c;         // Red instead of blue

// Change secondary colour
$secondary: #2ecc71;       // Green

// Change border radius
$border-radius: 0.75rem;   // More rounded corners
$border-radius-pill: 50rem;

// Change font family
$font-family-sans-serif: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;

// Change spacer (base for m-* and p-* utilities)
$spacer: 1.25rem;          // Default is 1rem

// Change body background and text
$body-bg: #f0f4f8;
$body-color: #2c3e50;

// Change heading font weight
$headings-font-weight: 700;

// Change link colour
$link-color: $primary;

// Change card border radius
$card-border-radius: $border-radius;

// Change input focus ring
$input-focus-border-color: $primary;

// ============================================================
// STEP 2 — Import Bootstrap source (AFTER your overrides)
// ============================================================
@import "../node_modules/bootstrap/scss/bootstrap";

// ============================================================
// STEP 3 — Add your own extra custom styles below
// ============================================================
.btn-custom {
  background: linear-gradient(135deg, $primary, darken($primary, 15%));
  color: white;
  border: none;

  &:hover {
    background: darken($primary, 10%);
    color: white;
  }
}

.hero-section {
  background: linear-gradient(135deg, $primary 0%, darken($primary, 20%) 100%);
  color: white;
  padding: 5rem 2rem;
  border-radius: $border-radius;
}

Importing only what you need (tree-shaking):
// Import only specific Bootstrap modules
@import "../node_modules/bootstrap/scss/functions";
@import "../node_modules/bootstrap/scss/variables";
@import "../node_modules/bootstrap/scss/mixins";

// Only include what your project uses:
@import "../node_modules/bootstrap/scss/reboot";
@import "../node_modules/bootstrap/scss/grid";
@import "../node_modules/bootstrap/scss/utilities";
@import "../node_modules/bootstrap/scss/buttons";
@import "../node_modules/bootstrap/scss/navbar";
@import "../node_modules/bootstrap/scss/modal";
// ... add more as needed
✅ Summary — The correct order:
1. Override $variables
2. @import "bootstrap"
3. Write your own custom styles

⚠️ Never edit Bootstrap's source files directly — override via your own custom.scss.
